        La salida no es exactamente igual a la del ejemplo, 
se logro hacer lo mas parecida posible.
        Ya el programa corre bajo el formato ./setlan, primero se debe hacer
chmod +x setlan.
       

        Lista de Flags:

        -t -----------------> imprime la lista de tokens
        -a -----------------> imprime el AST
        -s -----------------> imprime la tabla de simbolos
        -all ---------------> imprime todas la anteriores
       

        Formato de ejemplo:
        
           python setlan.py prueba.stl -t
        

        Archivos incluidos:

        setlan.py -----------> main
        setlan_lex.py -------> analizador lexicografico
        setlan_instr.py -----> conjunto de instrucciones de setlan
        setlan_exp.py -------> clases de las expresiones del lenguaje setlan
        setlan_fun.py -------> operadores del lenguaje setlan
        setlan_TablSim.py ------> generador de la tabla de simbolos
