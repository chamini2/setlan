README

En esta entrega se realizó la implementación del analizador lexicográfico para 
el lenguaje setlan proporcionado por los preparadores, para ello se utilizó el 
lenguaje python junto con el módulo lex de la herramienta PLY.

Tomando como referencia el ejemplo de PLY proporcionado por los preparadores,
se diseñaron diferentes identificadores por cada símbolo o palabra que admite 
el lenguaje, así como funciones para transformación de valores a sus respectivos
tipos, reconocimiento de comentarios y caracteres a ignorar. El analizador
lexicográfico lee una a una las palabras del archivo suministrado, le asigna el
identificador de token apropiado y lo agrega a una lista de tokens que será
impreso al terminar la lectura mientras no se encuentren caracteres inesperados.
En caso de encontrar uno o más caracteres inesperados, éstos serán agregados a 
una lista de errores que será impreso al terminar la lectura, ignorando la lista
de tokens.

Para ejecutar el programa se escribe por consola:

> chmod u+x ./setlan (para asignar permisos de ejecución, puede no ser necesario
		      en ciertos equipos)
> ./setlan arhivo.stl

donde:
	-setlan es el programa principal
	-archivo.stl es un código de lenguaje setlan a ser leído por el programa
